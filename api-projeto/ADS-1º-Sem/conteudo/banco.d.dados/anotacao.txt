-simples = nome completo, tipo do sençor
-composto = endereço , é dividido em partes de familhas difererntes como ... "rua" "bairro", que são de familias diferentes
-monovalorado = 
-multivalorado = valores divididos mas da mesma familia - por exemplo o telefone = pode ter telefone fixo, celular ... etc
-derivados= é um dado que passa por um processamento, por exemplo a idade que tem que ser atualizado a cada ano...
-indentificador= primary key , ou seja o atributo que indentifica a tabela, pode ser simples que é unica ou composta que depende de mais de um atributo.


entidade = tabelas

entidade forte= é ima tabela que não nescecita de outra tabela para existir
entidade fraca = depende de outra tabela  para existir
